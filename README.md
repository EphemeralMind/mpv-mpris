# mpv-mpris
MPRIS plugin for mpv written in C. Requires mpv to be built with `--enable-cplugins` (default as of mpv 0.26).

Implements `org.mpris.MediaPlayer2` and `org.mpris.MediaPlayer2.Player` D-Bus interfaces.
